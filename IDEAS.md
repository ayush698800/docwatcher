# DocDrift — Future Ideas

## v2.0 — Interactive TUI

User just types `docdrift` and gets an interactive menu:
- Check for stale docs
- Stage files
- Commit changes (with doc check built in)
- Push to remote
- Build doc index
- Toggle LLM on/off
- Settings

Arrow keys to navigate. No commands to memorize.
Makes DocDrift feel like a real developer tool not just a script.

## v2.0 — CascadeAI (separate product idea)

AI that auto-propagates code changes to dependent files.
You change one function, it ripples the change everywhere that calls it.
Name ideas: CascadeAI, RippleAI
